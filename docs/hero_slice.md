# Project Hex - Wow Slice Roadmap

## Phase 0 - Project Setup
- Create Godot 4.6 project
- Establish folder structure
- Create base test scene
- Add placeholder camera and lighting
- Add input actions

## Phase 1 - Player Feel
- 2.5D CharacterBody3D controller
- Left/right movement
- Jump
- Coyote time
- Jump buffer
- Dash
- Squash/stretch animation
- Landing effects

## Phase 2 - Room Construction
- Modular snapped room pieces
- Dark block material
- Emissive edge/glow material
- One hand-built test room
- Basic collision

## Phase 3 - Combat Feel
- Pulse blaster weapon
- Projectile or hitscan
- Hit flash
- Knockback
- Basic enemy health
- Enemy death burst

## Phase 4 - Enemy Feel
- Sharp red chaser enemy
- Floating shooter enemy
- Simple attack tells
- Damage to player

## Phase 5 - Visual Identity
- Swirly shader background
- Parallax layer
- Fog particles
- Foreground motes
- Platform edge glow
- Player/enemy emissive materials

## Phase 6 - Sound Pass
- Jump
- Land
- Dash
- Shoot
- Hit
- Enemy death
- Ambient loop

## Phase 7 - Trailer Room Polish
- Camera smoothing
- Screen shake
- Room intro/drop-in moment
- Enemy spawn sequence
- Room clear effect
- Final polish pass
