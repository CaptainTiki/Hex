# Project Hex - Concept

Project Hex is a 2.5D roguelike cyber-platformer built with 3D assets.

The player is a scientist/hacker deploying a small modular probe into a strange digital ecosystem inside a computer/network. The world is abstract, glowing, organic, and hostile. The player/probe is soft, rounded, blue/cyan, and squishy. Enemies are sharp, angular, red/orange, and dangerous.

## Current Goal

Build one trailer-ready room that proves the feel, style, movement, combat, shaders, particles, lighting, and sound direction.

## Not In Scope Yet

- procedural generation
- full roguelike run structure
- permanent progression
- multiple biomes
- full weapon customization
- large enemy roster
- room editor
